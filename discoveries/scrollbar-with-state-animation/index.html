Custom Native Scrollbars!

We all felt delighted to have our scrollbars styled with CSS. We no longer rely on hacky JavaScript scrollbar bloatware to get customized scrollbars. Though implementations vary across different browsers, ::-webkit-scrollbar pseudo-class and scrollbar-color are the most common practices among them.

A vigorous effect I really like of custom websites is that the scrollbars appear when we need them. When we hover over the content, there arrives a thin line indicating the progress. Oppositely, it gently fades out but remains scrollable. Try hover in and out over the document to see the effect.

How does it work?

There is no straightforward way to animate the color value of WebKit scrollbars though, there is a hack. That is to animate the color of the body and utilize color inheritance to apply to scrollbars. Also, there would be a wrapper element to allow a different text color to the content.Moreover, Firefox has got your back with scrollbar-color and scrollbar-width property. We can animate it through a transition declaration.
